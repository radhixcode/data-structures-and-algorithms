## Algorithms Complexities
### Sorting algorithms
- Bubble sort: O(n²)
- Insertion sort: O(n²)
- Selection sort: O(n²)
- Quick sort: O(n²)
- Merge sort: O(n log n)
- Heap sort: O(n log n)
- Radix Sort: O(nm)

### Searching algorithms
- Binary search: O(log2n)
- Depth first search: O(n)
- Breadth first search: O(n)
- Pre-order, in-order, post-order traversals: O(n)
